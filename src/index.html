Математические выражения

  • N! - факториал N
  • loga_N - логарифм N по основанию a
  • N**(1/a) - корень a степени из N
  • N**a - N в степени a
  • Nea = N*10**a

Горячие клавиши

  • Enter - вычислить
  • Ctrl + Z - на действие назад
  • Ctrl + Shift + Z / Ctrl + Y - на действие вперед
  • Space - фильтрация
  • Q - подсветка ошибок
  • R - строгий режим
